Was ist eine Website?

Eine website, webseite, homepage, internet seite 
ist ein Dokument mit informationen der online bereit gestellt würd im html oder text format.
Es ist unter eine url zu finden.

---

Was ist HTML?

- HTML steht für Hyper Text Markup Language.
- HTML beschreibt die struktur von eine Web page.
- HTML elemente sind bausteine eine HTML page.
- HTML elemente werden mit tags dargestellt um content zu zeigen.
- Browser zeigen nicht unsere tags.
